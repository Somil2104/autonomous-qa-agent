# Product Specifications

## Discount Codes
- The discount code SAVE15 applies a 15% discount to the subtotal.
- Only one discount code can be applied per purchase.
- Invalid or expired codes must show an error message.

## Shipping Costs
- Standard shipping is free.
- Express shipping costs $10.

## Cart
- Users can modify quantities on the checkout page.
- Total price must be recalculated dynamically when quantity changes.
